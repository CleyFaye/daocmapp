Dépôt "pour la postérité" d'un programme de cartographie pour DAoC (Dark Age of Camelot).

Pour l'histoire, à l'époque un mapper existait en Python (reproduit ici dans le répertoire MAPPER_M). Je l'ai réécrit "pour le fun" en Delphi, présent dans le répertoire DAOCMAPP.

Je n'ai plus aucune idée de comment ça fonctionne, mais ça reste du code "qui marchait" donc je laisse ça là, on va dire à des fins éducatives :D

Le mapper original en Python est aussi présent dans ce dépôt; à ma connaissance le site d'origine n'existant plus (et DAoC non plus d'ailleurs).
